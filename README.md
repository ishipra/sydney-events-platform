# Sydney Events Aggregation Platform

Full-stack MERN application that:

- Scrapes Sydney events automatically
- Stores in MongoDB Atlas
- Auto-updates with cron
- Supports Google OAuth login
- Admin dashboard with filtering & import system
- Lead capture before redirect

## Tech Stack

- React
- Node.js
- Express
- MongoDB Atlas
- Cheerio Scraping
- Google OAuth
- JWT
- node-cron

## Features

- Event status detection (new, updated, inactive, imported)
- Keyword & status filtering
- Protected dashboard
- Email consent capture
